Hierarchical Task Analysis (HTA)


Colour scheme

The color scheme is largely dependant on a monochromatic range of grays, with a large emphasis on warmness. The monochroatic theme was selected due to the emphasizing ability it has, which allows for the user's focus to remain on the content highlighted by the brighter color (#efefef) in the center. The warmness reduces the strain on the users eyes which increases the overal experience. The maroon color is evident only on the main page to add some interest in the page. Non-monochromatic colors are not used elsewhere due to the presence of the multi-colored team logos.
